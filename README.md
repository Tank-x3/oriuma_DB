# 🐎 Oriuma DB (オリウマ キャラクターデータベース)

**「オリウマ」のキャラクター設定を、みんなで共有・保存できるデータベースツールです。**

[👉 データベースを開く (Demo Link)](https://your-username.github.io/oriuma-db/)  
*(ここに実際のGitHub PagesのURLを入れてください)*

---

## 📖 概要
このツールは、オリウマの設定やプロフィールを登録し、一覧として閲覧できるようにするために開発されました。

PC・スマートフォンの両方に対応しており、ダークモードも搭載しています。

## 🚀 基本的な使い方

### 1. オリウマを見る (List)
トップページまたはメニューの「List」から、登録されているオリウマの一覧を見ることができます。
カードをクリックすると、詳細なプロフィール（誕生日、身長/体重/スリーサイズ、学年、所属寮、補足情報など）が表示されます。

### 2. オリウマを登録する (Register)
「Register」ページから、あなたのオリウマを登録できます。
*   **基本情報:** 名前（カナ/欧字/香港名）、トレーナー名、アイコン画像URLなど。
*   **詳細データ:** 身長、スリーサイズ、学年など、基本的な項目があらかじめ用意されています。
*   **自由記述:** 定義されていない項目も「自由項目」として追加可能です。ジャンルのリクエストはAboutページから総合お問い合わせフォームにリクエストを送ってください。

> **⚠️ 重要:**  
> 登録時に設定する「編集用パスワード」は、後で内容を変更する際に必要になります。忘れないようにしてください。  
> パスワードは暗号化されてDBに記録されます。問い合わせがあっても調べることができません。  
> もし忘れてしまった場合は、後述の説明、およびAboutページの説明に従って操作してください。

### 3. 情報を更新する (Edit)
詳細画面の「📝 編集する」ボタンから、登録済みデータの修正、追記ができます。
編集には登録時に設定したパスワードが必要です。

---

## ✨ 便利な機能

### 🌙 ダークモード対応
画面右上の「🌙」ボタンを押すと、目に優しいダークモードに切り替わります。夜間の作業や閲覧に最適です。

### 📂 バックアップと復元
登録・更新完了時に、キャラクターデータを **JSONファイル（バックアップ）** としてダウンロードできます。
このファイルを持っていれば、もしパスワードを忘れてしまっても「新規登録画面」で読み込むことで、データを復元して再登録（作り直し）することが簡単にできます。

### 🆘 パスワードを忘れた場合
編集画面から「パスワードを忘れた場合はこちら」を選択することで、管理者へリセット依頼やデータ統合の依頼を送る機能があります。
